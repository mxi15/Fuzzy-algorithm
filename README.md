# Fuzzy-algorithm
模糊算法

单纯的使用模糊计算算出系统运转时间与系统温度、湿度之间的相关性，再去模糊计算得到具体的系统运作时间。<br>

题目：
计算出下列温度、湿度值对应的运转时间。<br>

序号|温度|湿度|运转时间
--|--|--|--
1|56|33|
2|28|20|
3|18|40|
4|80|50
2|70|10|
